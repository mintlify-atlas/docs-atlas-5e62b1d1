Click-n-Color offers extensive customization options through its settings panel, allowing you to tailor the extension to your workflow and visual preferences.

## Settings overview

Access settings by clicking the **Settings** button (gear icon) in the popup. The settings are automatically saved and persist across sessions.

## Default color format

Choose which color format is selected by default when you open the extension.

### Available formats

- **RGB**: `rgb(255, 0, 128)` - Red, Green, Blue values
- **HEX**: `#ff0080` - Hexadecimal color code
- **HSL**: `hsl(328, 100%, 50%)` - Hue, Saturation, Lightness

```javascript
// popup.js:27
const ALLOWED_FORMATS = ['rgb', 'hex', 'hsl'];
```

### Format conversion

The extension converts between formats on-the-fly when you switch format chips:

```javascript
// popup.js:294-304
function updateColorValueFormat() {
    const currentColor = getComputedStyle(colorPreview).backgroundColor;
    const format = document.querySelector('.format-chip.active').textContent;
    if (format === 'RGB') {
        colorValue.textContent = currentColor;
    } else if (format === 'HEX') {
        colorValue.textContent = rgbToHex(currentColor);
    } else if (format === 'HSL') {
        colorValue.textContent = rgbToHsl(currentColor);
    }
}
```

<Accordion title="RGB to HEX conversion">
  ```javascript
  // popup.js:306-314
  function rgbToHex(rgb) {
      const match = rgb.match(/rgb\((\d+),\s*(\d+),\s*(\d+)\)/);
      if (!match) return rgb;
      const r = parseInt(match[1]);
      const g = parseInt(match[2]);
      const b = parseInt(match[3]);
      return `#${((1 << 24) + (r << 16) + (g << 8) + b).toString(16).slice(1)}`;
  }
  ```
</Accordion>

<Accordion title="RGB to HSL conversion">
  ```javascript
  // popup.js:316-341
  function rgbToHsl(rgb) {
      const match = rgb.match(/rgb\((\d+),\s*(\d+),\s*(\d+)\)/);
      if (!match) return rgb;
      let r = parseInt(match[1]) / 255;
      let g = parseInt(match[2]) / 255;
      let b = parseInt(match[3]) / 255;
      const max = Math.max(r, g, b);
      const min = Math.min(r, g, b);
      let h, s, l = (max + min) / 2;
      if (max === min) {
          h = s = 0; // achromatic
      } else {
          const d = max - min;
          s = l > 0.5 ? d / (2 - max - min) : d / (max + min);
          switch (max) {
              case r: h = (g - b) / d + (g < b ? 6 : 0); break;
              case g: h = (b - r) / d + 2; break;
              case b: h = (r - g) / d + 4; break;
          }
          h = Math.round(h * 60);
      }
      s = Math.round(s * 100);
      l = Math.round(l * 100);
      return `hsl(${h}, ${s}%, ${l}%)`;
  }
  ```
</Accordion>

## Maximum palette size

Control how many colors can be stored in your palette.

- **Minimum**: 10 colors
- **Maximum**: 100 colors
- **Default**: 50 colors

When the limit is reached, the oldest colors are automatically removed as new ones are added.

```javascript
// popup.js:45-48
const parsedPaletteSize = parseInt(safeSettings.maxPaletteSize, 10);
const maxPaletteSize = Number.isFinite(parsedPaletteSize)
    ? Math.min(100, Math.max(10, parsedPaletteSize))
    : 50;
```

<Note>
  The palette size is validated to ensure it stays within the 10-100 range. Invalid values automatically reset to the default of 50.
</Note>

## Auto-copy to clipboard

Enable or disable automatic copying of picked colors to your clipboard.

- **Enabled** (default): Colors are automatically copied when picked
- **Disabled**: You must manually click the copy button

```javascript
// popup.js:349-353
if (settings.autoCopy) {
    copyTextToClipboard(colorValue.textContent);
}
```

The auto-copy feature uses the currently selected color format, so if you have HEX selected, the HEX value will be copied.

```javascript
// popup.js:117-127
async function copyTextToClipboard(text, successMessage = 'Color copied to clipboard!') {
    try {
        await navigator.clipboard.writeText(text);
        showNotification(successMessage);
        return true;
    } catch (error) {
        console.error('Clipboard write failed:', error);
        showNotification('Clipboard copy failed. Check permissions.');
        return false;
    }
}
```

## Saving settings

Click the **Save Settings** button to apply your changes:

```javascript
// popup.js:506-527
function saveSettings() {
    const defaultFormat = document.querySelector('input[name="defaultFormat"]:checked').value;
    const parsedPaletteSize = parseInt(document.getElementById('maxPaletteSize').value, 10);
    const maxPaletteSize = Number.isFinite(parsedPaletteSize)
        ? Math.min(100, Math.max(10, parsedPaletteSize))
        : 50;
    document.getElementById('maxPaletteSize').value = maxPaletteSize;
    const autoCopy = document.getElementById('autoCopy').checked;
    settings = {
        defaultFormat,
        maxPaletteSize,
        autoCopy
    };
    chrome.storage.local.set({ settings }, () => {
        settingsModal.style.display = 'none';
        showNotification('Settings saved!');
        // Apply default format immediately
        activateFormatChip(defaultFormat.toUpperCase());
        updateColorValueFormat();
    });
}
```

### Applying themes

Themes are applied by updating CSS custom properties on the document root:

```javascript
// popup.js:544-567
function applyTheme(theme) {
    const root = document.documentElement;
    // Reset to default theme
    root.style.setProperty('--primary', '#8a2be2');
    root.style.setProperty('--secondary', '#00bfff');
    root.style.setProperty('--accent', '#ff1414');
    root.style.setProperty('--background', '#121212');
    root.style.setProperty('--text', '#ffffff');
    root.style.setProperty('--card', '#1e1e1e');
    root.style.setProperty('--hover', '#f5e0e0');
    // Apply theme-specific colors
    if (theme === 'light') {
        // ... light theme overrides
    } else if (theme === 'blue') {
        // ... blue theme overrides
    }
}
```

### Saving themes

Select your preferred theme and click **Save Theme Settings**:

```javascript
// popup.js:532-540
function saveThemeSettings() {
    const selectedTheme = sanitizeTheme(document.querySelector('input[name="theme"]:checked').value);
    chrome.storage.local.set({ theme: selectedTheme }, () => {
        themesModal.style.display = 'none';
        showNotification('Theme saved!');
        applyTheme(selectedTheme);
    });
}
```

## Settings validation

All settings are sanitized and validated before being saved:

```javascript
// popup.js:39-59
function sanitizeSettings(rawSettings) {
    const safeSettings = rawSettings && typeof rawSettings === 'object' ? rawSettings : {};
    const defaultFormat = ALLOWED_FORMATS.includes(safeSettings.defaultFormat)
        ? safeSettings.defaultFormat
        : 'rgb';

    const parsedPaletteSize = parseInt(safeSettings.maxPaletteSize, 10);
    const maxPaletteSize = Number.isFinite(parsedPaletteSize)
        ? Math.min(100, Math.max(10, parsedPaletteSize))
        : 50;

    const autoCopy = typeof safeSettings.autoCopy === 'boolean'
        ? safeSettings.autoCopy
        : true;

    return {
        defaultFormat,
        maxPaletteSize,
        autoCopy
    };
}
```

<Tip>
  Settings are persisted in Chrome's local storage and will be restored even after restarting the browser or updating the extension.
</Tip>

## Notifications

Whenever settings are saved, a confirmation notification appears for 2 seconds:

```javascript
// popup.js:238-249
function showNotification(message) {
    notification.textContent = message;
    notification.classList.add('show');

    if (notificationTimeoutId) {
        clearTimeout(notificationTimeoutId);
    }

    notificationTimeoutId = setTimeout(() => {
        notification.classList.remove('show');
    }, 2000);
}
```
